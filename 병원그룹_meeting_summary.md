# 병원그룹 미팅 서머리

> 기간: 2026-03-25 (1~3차 미팅)
> 참석: CTO, 의료정보팀장, 의료정보보안팀장, IRB 담당자

## 논의 내용

### 1. 현재 과제와 목표
- 영상의학과 판독문 검색 시 의사마다 표현이 달라 원하는 결과를 찾기 어려운 상황
- 간호기록 인수인계에 수기로 약 30분 소요 — 효율화 필요
- CTO: "이번에는 우리 데이터를 기반으로 직접 해보고 싶다"

### 2. 기술 환경 확인
- EMR은 자체개발, 온프레미스 서버에서 운영 중
- 영상 시스템(PACS)은 Infinitt 제품, 표준 연동(HL7 FHIR) 가능
- GPU 서버(A100 x2)를 연구팀과 공유하여 활용 가능

### 3. 데이터 및 규제 검토
- 개인건강정보는 가명처리(이름/생년월일/주민번호 마스킹) 후 활용 가능 — 내부 검토 완료
- IRB 간편심의로 진행 가능, 약 2주 소요 예상
- AI 결과를 의사가 최종 확인하는 보조 도구 형태로 운영

### 4. 작년 경험
- 외부 업체 챗봇 시범 운영 → 의학용어 이해 부족으로 정확도 미달, 중단
- 이번에는 자체 데이터 기반 + 의학용어 매핑으로 차별화

## 합의 사항

- **파일럿 범위**: 본원 영상의학과 판독문 검색 보조
- **예산**: 5천만원 이내 (파일럿 기준)
- **시작 시점**: 4월 중순 목표 (IRB 심의 2주 + 데이터 준비 2주)
- **내부 인력**: 의료정보팀 3명 투입
- **운영 방식**: 의사/간호사 전용, 환자 직접 노출 없음
- **의사결정**: 의료정보위원회에서 AI 관련 안건 심의

## 참고

- 간호기록 요약(교대 인수인계용)은 2순위로, 파일럿 성공 후 확대 검토
- 분원 확대는 본원 파일럿 완료 후 별도 논의
- 장기적으로 환자 설명용 활용 가능성도 열어둔 상태
