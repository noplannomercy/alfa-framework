## 예상 결과

### 행위 유형 분포
- 탐색: 60% — 유사 소견/사례 검색, 동의어 확장
- 접근: 25% — 특정 환자/기간/모달리티 판독문 조회
- 실행(유연): 10% — 소견 비교, 참고 자료 정리
- 실행(확정): 5% — 포맷 리포트

### GraphRAG 필요: **예**
- 의학용어 동의어/유사표현 매핑이 핵심
- 이전 PoC 실패 원인을 직접 해결하는 요소

### 예상 Phase 0 소요: **2주**
- Week 1: 판독문 샘플 데이터 분석, 용어 불일치 패턴 정량화, 리소스 인벤토리 확정
- Week 2: 행위 유형 분포 검증, 기술 스택 확정, Phase 1 상세 계획 수립

## 투입 권고

**즉시 Phase 0 착수 권고.**

IRB 심의(2주)와 데이터 준비(2주)를 병행하면서 Phase 0을 동시 진행할 수 있음.
Phase 0 완료 시점(~2주)에 IRB 통과가 맞물리면, 바로 Phase 1(PoC 구현)에 진입 가능.

```
3월 말          4월 초          4월 중순         6월 말
  │               │               │               │
  ├─ IRB 접수 ────┤               │               │
  ├─ Phase 0 ─────┤               │               │
  │               ├─ 데이터 준비 ──┤               │
  │               │               ├─ Phase 1 PoC ─┤
  │               │               │               ├─ 평가
```
