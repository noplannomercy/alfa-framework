## ALFA Phase 진행 상태

```
Phase 0 WHY:      ✅ 완료 — CTO 승인, 월 200만원, 한 달
Phase 1 ASSESS:   ✅ 완료 — 인프라 있음, 인재(개발자 조직), 규제 가벼움
Phase 2 GOVERN:   ✅ 완료 — 고객사 정보 제외 원칙만. 필수 체크리스트 대부분 해당 없음
Phase 3 PLATFORM: ✅ 거의 완료 — RDS PostgreSQL + pgvector + Bedrock. 인프라 새로 구축할 게 없음
Phase 4 PILOT:    🔄 바로 진입 가능
```

> **Phase 0~3이 미팅 1회에서 전부 끝남.** 거버넌스가 가벼운 업종의 ALFA 속도.

## 기술 메모

- Confluence 500페이지 → Docling으로 파싱 가능 (HTML/마크다운)
- RDS PostgreSQL에 pgvector → 추가 벡터DB 비용 없음
- 슬랙 #dev-help → 2차 데이터 소스. 노이즈 필터링 필요 (잡담 제거)
- "배포 에러" 검색 시 수십 개 히트 → 메타데이터 필터링 (서비스명, 도구명) 또는 리랭킹 필요
- GraphRAG 필요성: 낮음. 개발 문서는 용어 충돌이 적고 질의가 구체적
