## 핵심 결정 사항

**1. 온프레미스 전용**
- 데이터 외부 반출 금지 → 클라우드 API 사용 불가
- A100 서버에 로컬 LLM (Ollama) 설치
- 데이터가 사내 서버 밖으로 나가지 않으므로 DLP 자연 충족

**2. 데이터 분류 불필요**
- 전부 사내 품질 문서 + 사내 품질 DB
- 고객 개인정보 없음
- 분류 없이 전부 LLM 투입 가능

**3. 거버넌스 극경량**
- 공장장 승인 + 최차장 실행의 2인 구조
- 별도 컴플라이언스/법무 검토 불필요
- 현대캐피탈(미팅 2회, 체크리스트 6개)과 대비되는 케이스
