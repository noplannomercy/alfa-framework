---

## 평가 요약

**강점:**
- A100 GPU + PostgreSQL 이미 보유 → 추가 비용 0
- 공장장 직접 지시 → 스폰서 명확
- 불량 이력 만 건 + 문서 150개 → 데이터 충분

**약점:**
- AI 전문 인력 없음 → 외부 의존
- 온프레미스 필수 → 로컬 LLM 세팅에 시간 필요
- 불량 이력 표현 비통일 → 검색 품질 이슈 예상

**위험:**
- 온프레미스 로컬 LLM이 API 대비 성능이 낮을 수 있음 → 모델 선정 중요
- 불량 표현 비통일 → 기본 RAG로 한계 시 GraphRAG 필요
