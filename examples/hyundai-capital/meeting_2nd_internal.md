## 기술 메모

- PoC 동음이의어 문제 ("한도"가 할부/개인대출에서 다른 의미) → **GraphRAG 승격 시그널**
- 데이터 분류: 기존 개인정보 3단계에 LLM 투입 여부 추가 → 문서화 필요
- 파일럿 대상: 상담팀이 최다 사용부서 → 효과 측정 용이

## ALFA Phase 진행 상태

```
Phase 0 WHY:    ✅ 완료 — 목적, KPI, 스폰서십, 예산, 일정 확인
Phase 1 ASSESS: ✅ 완료 — 인프라/데이터/규제/인재/문화 전부 확인
Phase 2 GOVERN: ✅ 필수 항목 확인 완료 — 문서화만 남음
Phase 3 PLATFORM: 🔄 진입 — Bedrock 개통 대기, 아키텍처 확정
Phase 4 PILOT:   ⬜ 대기 — 유즈케이스 확정 (사내 문서 검색), IRM 투입 시점
Phase 5 SCALE:   ⬜ 대기
Phase 6 OPERATE: ⬜ 대기
```
