# ALFA 고객 프로필 — 현대캐피탈

> 작성 기준: 1차 미팅 (2026-03-25) + 2차 미팅 (2026-03-28) 결과
> 상태: 확정 — ALFA Phase 0 입력 가능

---

## [1] 회사 개요

- **회사명:** 현대캐피탈
- **업종:** 금융 (여신전문금융업)
- **규모:** 대기업
- **주력 사업:** 자동차 할부/리스, 개인금융(대출), 기업금융
- **직원 수:** (확인 필요)

---

## [2] 현재 AI/LLM 상태

- **AI/LLM 도입 경험:** PoC 경험 (1년 전)
- **이전 시도:**
  - 뭘 했는가: 사내 매뉴얼/규정집 PDF를 Qdrant 벡터DB에 넣고 RAG 검색
  - 왜 중단했는가: 동음이의어 정확도 이슈 ("한도"가 할부/개인대출에서 혼재) + LLM 기술 급변으로 방향 재설정
  - 남아있는 자산: 매뉴얼/규정집 PDF 약 200개 (공유폴더)
    - 대출 상품 매뉴얼 (최다), 자동차 할부/리스 안내서, 내부 규정집, 업무 절차서
    - 전부 내부 문서 (고객 정보 없음)

---

## [3] 인프라 현황

- **클라우드:** AWS (확정)
- **기존 환경:** VPC + 보안 그룹 세팅됨
- **LLM 서비스:** Bedrock (개통 신청 중)
- **기존 DB:** (확인 필요)
- **네트워크 제약:** VPC 내 프라이빗 서브넷, 인터넷 경유 없음
- **보안 등급:** 금융

---

## [4] 규제 환경

- **적용 규제:**
  - 인공지능기본법 (2026.01 시행) — 내부 직원용은 고영향AI 해당 없음. 투명성 + 안전성 경량 대응
  - 금감원 AI 가이드라인 7대 원칙 — **보조수단성** 핵심 (AI는 보조, 최종 판단은 사람)
  - 개인정보보호법 — 고객 개인정보/신용정보 LLM 투입 금지
- **감독 기관:** 금감원
- **컴플라이언스 담당:** 박과장 (컴플라이언스팀)
- **파일럿 범위 규제 이슈:** 없음 확인 (2차 미팅)
- **데이터 분류:** 기존 개인정보 3단계(일반/중요/민감)에 LLM 투입 가능 여부 추가
  - 일반(공개): 상품 정보, 공시 자료 → LLM 투입 가능
  - 중요(내부): 매뉴얼, 규정, 업무 절차 → LLM 투입 가능 (RAG 대상)
  - 민감: 고객 개인정보, 신용정보 → **LLM 투입 금지**

---

## [5] 하고 싶은 것

- **경영진 지시:** "LLM 거버넌스 구축해라" — 체계 + 활용
- **현업 니즈:** 사내 문서 검색 속도 개선
  - 현재: 사내 포털 키워드 검색 → 잘 안 됨 → 공유폴더/동료 문의
  - 병목: 특히 상담사가 고객 통화 중 문서 찾을 때 시간 과다 소요
- **1차 파일럿 확정:** 사내 지식 검색 (내부 직원용)
- **2차 후보:** 고객 상담 보조 (거버넌스 강화 후)
- **장기:** 여신 심사 보조 등

---

## [6] 제약 사항

- **예산:** 월 500만원 이내 (API 비용). 인프라 투자(GPU 등)는 파일럿 결과 후 판단
- **일정:** 3~4개월 (3개월 파일럿 + 4개월차 결과 보고)
- **내부 인력:** 김대리 (IT기획팀). AI 전문 인력 제한적
- **외부 파트너:** 아키텍트 + 컨설팅 + 구축 풀코스
- **고객 대면 업무:** 1차 파일럿 제외 (금감원 리스크)
- **파일럿 참여자:** IT팀 10명 (기술 피드백) + 상담팀 40명 (실사용 피드백)

## 로드맵 합의

| 단계 | 기간 | 내용 |
|------|------|------|
| 1단계: 거버넌스 확정 | 1~2주 | 필수 항목 문서화, Bedrock 개통 |
| 2단계: 인프라 세팅 | 2~3주 | Bedrock + 벡터DB + 가드레일 + 데이터 파이프라인 |
| 3단계: 파일럿 | 4~6주 | 사내 문서 검색, IT 10명 + 상담팀 40명, 2주 운영 |
| 4단계: 판정+확대 | 2~4주 | 평가 → Go/No-Go → 전사 오픈 + 2차 유즈케이스 |

## 기술 메모 (내부)

- PoC 동음이의어 문제 ("한도" 혼재) → **GraphRAG 승격 시그널**. IRM Phase 1-A에서 승격 판단 필요
- 데이터 200개 전부 내부 등급 → 데이터 파이프라인 설계 단순
- 상담팀이 최다 사용부서 → 파일럿 효과 측정 용이
