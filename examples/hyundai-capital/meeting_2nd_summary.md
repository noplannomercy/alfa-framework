## 논의 내용

### 1. 거버넌스 필수 항목 확인

| # | 항목 | 결과 |
|---|------|------|
| 1 | AI 의사결정 기구 | IT거버넌스 위원회에 AI 안건 편입 |
| 2 | RACI | IT기획팀 실행 / 컴플라이언스팀 검토 / 위원회 승인 |
| 3 | 데이터 분류 | 기존 개인정보 3단계(일반/중요/민감)에 LLM 투입 가능 여부 추가. 민감 투입 금지 |
| 4 | 접근제어 | AWS IAM 기반 부서별 권한 분리 (기술 영역) |
| 5 | 금감원 보조수단성 | 문서 검색 → 자연 충족 (직원이 판단). 고객 대면 시 재검토 |
| 6 | 인공지능기본법 | 내부 직원용 → 고영향AI 해당 없음. 투명성(AI 표시) + 안전성(가드레일) 경량 대응 |

**컴플라이언스팀 확인: 파일럿 범위에서 규제 이슈 없음**

### 2. 예산 & 일정

| 항목 | 내용 |
|------|------|
| 예산 | 월 500만원 이내 (API 비용). 인프라 투자는 결과 후 판단 |
| 일정 | 3~4개월 (3개월 파일럿 + 4개월차 결과 보고) |

### 3. 파일럿 구성

| 항목 | 내용 |
|------|------|
| 대상 | 사내 문서 검색 (내부 직원용) |
| 참여자 | IT팀 10명 (기술 피드백) + 상담팀 40명 (실사용 피드백) |
| 기간 | 2주간 운영 후 평가 |

### 4. 로드맵 합의

| 단계 | 기간 | 내용 |
|------|------|------|
| 1단계: 거버넌스 확정 | 1~2주 | 필수 항목 문서화, Bedrock 개통 |
| 2단계: 인프라 세팅 | 2~3주 | Bedrock + 벡터DB + 가드레일 + 데이터 파이프라인 |
| 3단계: 파일럿 | 4~6주 | 사내 문서 검색, IT 10명 + 상담팀 40명, 2주 운영 |
| 4단계: 판정+확대 | 2~4주 | 평가 → Go/No-Go → 전사 오픈 + 2차 유즈케이스 |
